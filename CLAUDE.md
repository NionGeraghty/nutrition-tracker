# Nutrition Tracker

## Purpose
Learning project — full stack with Node/Express/TypeScript backend,
Next.js/React frontend, Postgres, Vitest/Jest, React Testing Library.

## How to work with me
- I'm learning this stack. Only prior experience: React.
- Explain your reasoning before writing non-trivial code — I need to
  understand the code, not just have it work.
- Prefer clarity over cleverness.
- Don't add libraries, patterns, or abstractions I haven't asked about
  without flagging it and explaining why first.
- If I ask you to build something, prefer smaller steps I can follow
  over one large multi-file change.
